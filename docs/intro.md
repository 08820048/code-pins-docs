# CodePins – IntelliJ 插件教程

本教程旨在详细介绍 CodePins 插件的所有功能，帮助您快速上手并高效使用。

## 插件概述

> CodePins 是一个轻量级的图钉系统插件，帮助开发者高效组织和导航代码。

**核心功能：**

- "📌" 将任何代码行或代码块标记为书签
- 支持备注、导航、搜索和标签筛选
- 提供删除、清空、导入/导出和分享功能
- 支持排序、拖拽排序、复制和刷新操作

**使用场景：**

- 临时代码笔记
- 错误位置跟踪
- TODO 任务管理
- 代码导航和书签

## 快速开始

### 添加您的第一个图钉

1. 在 IDE 中打开任何代码文件
2. 将光标移动到您想要标记的行
3. 按下 `Alt + P` (Windows/Linux) 或 `Option + P` (Mac)
4. 或者点击边框区域的图钉图标

您的第一个图钉现在已经创建完成！您可以在 CodePins 工具窗口中看到它。

### 图钉管理

CodePins 工具窗口提供了各种管理图钉的方式：

1. **查看图钉**
   - 打开 CodePins 工具窗口（`视图 -> 工具窗口 -> CodePins`）
   - 所有图钉将以列表形式显示
   - 点击任何图钉即可导航到其位置

2. **编辑图钉**
   - 双击图钉可编辑其备注
   - 右键点击可查看更多选项
   - 拖放可重新排序图钉

3. **搜索和筛选**
   - 使用搜索框查找特定图钉
   - 按标签或文件筛选图钉
   - 按不同标准排序图钉

### 添加备注和标签

每个图钉可以有：
- 描述性备注
- 多个标签
- 自定义颜色

编辑图钉的方法：
1. 在工具窗口中双击图钉
2. 在文本字段中添加您的备注
3. 使用标签选择器添加标签
4. 点击保存或按回车键

### 图钉组织

CodePins 提供了多种组织图钉的方式：

1. **标签**
   - 创建自定义标签
   - 为每个图钉分配多个标签
   - 按标签筛选图钉

2. **排序**
   - 按创建时间
   - 按文件名
   - 按备注内容
   - 自定义顺序（拖放）

3. **分组**
   - 按文件
   - 按标签
   - 按项目

### 分享和导出

与团队成员分享您的图钉：

1. **导出选项**
   - 导出为 JSON
   - 导出为 Markdown
   - 导出为图片

2. **导入选项**
   - 从 JSON 文件导入
   - 从剪贴板导入

3. **分享功能**
   - 分享单个图钉
   - 分享图钉集合
   - 带水印分享

## 故障排除

### 常见问题

**Q: 如果图钉位置不匹配或偏移怎么办？**
A: 这通常是由于代码变更导致的。尝试以下解决方案：
1. 使用工具窗口中的刷新按钮
2. 重新标记受影响的行
3. 将插件更新到最新版本

**Q: 如何在版本控制中处理图钉？**
A: 图钉默认存储在本地。要与团队分享：
1. 将图钉导出到文件
2. 将文件添加到版本控制
3. 团队成员可以导入图钉
